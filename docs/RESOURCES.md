# Recommended Learning Resources

## Documentation

* [Official GDScript Reference](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/index.html)

## Communication Channels

* [GDScript Discussions on Official Godot Forum](https://forum.godotengine.org/tag/gdscript)
* [GDScript Discussions on (Unofficial) Godot Forums](https://godotforums.org/t/gdscript)
* [StackOverflow questions tagged [gdscript]](https://stackoverflow.com/questions/tagged/gdscript) can be used to search for your problem and see if it has been answered already. You can also ask and answer questions.
* [r/godot on Reddit](https://www.reddit.com/r/godot/) is a subreddit where GDScript questions are relevant.
* [Official Godot Discord](https://discord.com/invite/4JBkykG)
